## 주요 변경 사항

- 수어 번역 모드 `ON_DEVICE`에서도 온라인이면 서버 STT를 사용하도록 변경
- STT 경로 판단을 수어 번역 모드가 아닌 네트워크 상태 기준으로 분리
- 온라인 STT는 기존처럼 백엔드 `/stt-config` 설정에 따라 REST/STREAMING/AUTO 사용
- 오프라인 상태에서는 로컬 STT fallback 유지

## 메모

- 수어 번역 기본값은 `ON_DEVICE`로 유지됩니다.
- STT는 수어 번역 모드와 분리되어, 온라인이면 서버 STT 기본값인 REST를 사용합니다.
- 백엔드 `TRANSLATION_STT_MODE`가 `STREAMING`/`AUTO`로 설정된 경우에만 해당 STT 경로를 탑니다.

## 주요 변경 사항
`5줄 이하로 유지`
- 소통 종료 후 분석 진행 중(`PENDING/PROCESSING`)이면 홈에서 일정 시간 자동 재조회하도록 개선
- 아이 발화가 없는 세션은 분석 생성 및 홈/리포트 집계에서 제외
- 소통 분석 세션 목록을 3개 단위 페이지 형태로 표시하고, 날짜/페이지 변경 시 토글 접힘 상태로 초기화
- 분석 상세의 가이드 항목이 비어 있을 경우 부모님용 기본 가이드 문구를 보완 표시
- 기존 소통 분석 조회에서 `EMPTY` 분석 상태는 제외

## 스크린샷/로그

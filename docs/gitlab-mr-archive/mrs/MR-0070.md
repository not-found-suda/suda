## 주요 변경 사항
- `AppEntryRoute`를 앱 시작 진입점으로 추가하고 인증 상태별 라우팅 분기 구현
- `AuthSessionManager`와 `AuthState`를 추가해 토큰/활성 childId 기반 세션 상태 판단
- `ActiveChildStorage`를 DataStore 기반으로 분리해 활성 childId 저장소 뼈대 구현
- 로그인/아이 선택/홈 placeholder route 추가
- 변경하지 않은 것 / 스코프 아웃: 실제 로그인 API, 아이 프로필 API, `/status` 토큰 검증, 화면별 개별 보호 가드는 제외

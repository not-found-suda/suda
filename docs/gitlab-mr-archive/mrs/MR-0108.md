### nej0720 at 2026-05-07T20:55:11.522+09:00 (comment)

```text
**\[P2\] 현재 문제가 아닌 questionId도 답변 제출할 수 있습니다**

답변 제출 로직에서 `questionId`가 해당 세션에 속하는지와 이미 답변된 문제인지만 확인하고 있습니다. 그래서 클라이언트가 현재 문제를 풀기 전에 뒤쪽 문제 ID를 보내면, 순서를 건너뛰고 먼저 채점/answered 처리될 수 있을 것 같습니다.

API 흐름이 `/questions/current`로 현재 문제를 조회한 뒤 순서대로 답변하는 방식이라면, 제출된 문제가 현재 미답변 첫 문제인지도 같이 확인해주면 더 안전할 것 같습니다!
```

**\[P1\] 세션/아이 프로필 소유자 검증이 없어 다른 사용자의 퀴즈에 접근할 수 있습니다**

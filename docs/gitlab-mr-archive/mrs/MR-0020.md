### hh001204 at 2026-04-25T19:11:58.307+09:00 (comment)

```text
P3: `.editorconfig` 설정 기준을 조금 정리하면 좋을 것 같습니다.

현재 루트 `.editorconfig`에도 kt 설정이 있고, `mobile/.editorconfig`에도 kt 설정이 추가되어 있습니다. 그런데 `mobile/.editorconfig`에 `root = true`가 있어 mobile 쪽은 루트 설정을 상속받지 않게 됩니다.

mobile에서 별도로 관리할 거라면 루트의 kt 설정을 정리하고, 루트에서 공통 관리할 거라면 mobile 쪽은 필요한 설정만 override하는 식으로 기준을 맞추면 좋을 것 같습니다.

그리고 `ktlint_function_naming_ignore_when_annotated_with = Composable`이 있으니, function-naming 규칙 전체를 끄는 `ktlint_standard_function-naming = disabled`는 제거해도 될 것 같습니다.
```

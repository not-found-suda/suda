## Description

## 요약
- 수어 번역이 서버/AUTO 경로를 타지 않도록 기본 번역 모드를 ON_DEVICE로 변경


## 작업 배경
- 관련 이슈: #S14P31A404-278
- 참고 문서/기획:
  - develop 반영 후 수어 번역 결과가 376번 시연 검증 브랜치와 다르게 동작하는 현상 확인
  - 수어 번역은 온디바이스 Qwen/rule 기반 경로를 유지하고, STT의 CLOVA 적용은 별도 분리 필요


## 주요 변경 사항
- `TranslationMode.DEFAULT`를 `AUTO`에서 `ON_DEVICE`로 변경
- `ConversationViewModel`의 초기 번역 모드를 `ON_DEVICE`로 변경
- 시연용 rule 기반 수어 번역 및 온디바이스 번역 경로가 기본으로 사용되도록 조정
- 스코프 아웃: STT 전용 모드 분리 및 CLOVA/AUTO STT 구조 개선은 후속 작업으로 분리


## 스크린샷/로그
- UI 변경 없음
- 실기기에서 수어 문장 인식 후 ON_DEVICE 번역 경로 동작 확인 예정

## 요약

* 학습 기능의 난이도 조회 API를 추가하고 단어장 조회 정렬 방식을 랜덤에서 순차 정렬로 변경

## 작업 배경

* 관련 이슈: #S14P31A404-52
* 참고 문서/기획:

  * 학습 기능 API 명세서
  * CODA 아동 대상 단어장 학습 기능 기획

## 주요 변경 사항

* `/api/v1/learn/levels` 난이도 목록 조회 API 추가
* `LearnDifficulty` enum에 난이도 표시명(displayName) 추가
* 단어장 조회 쿼리를 `RANDOM()` 기반에서 `sort_order ASC` 기반 순차 정렬로 변경
* Swagger Docs(`LearnControllerDocs`)에 난이도 조회 API 명세 추가

## 스크린샷/로그

* GET `/api/v1/learn/levels`

```json
[
  { "code": "EASY", "name": "쉬움" },
  { "code": "MEDIUM", "name": "보통" },
  { "code": "HARD", "name": "어려움" }
]
```

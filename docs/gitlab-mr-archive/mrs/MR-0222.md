## 주요 변경 사항
- 학습 퀴즈 결과 카드에 실제 인식된 발화(`recognizedText`)를 노출하고, 별/결과 카드 레이아웃을 개선
- 무음은 물론, `별 1개` 중 완전히 무관한 발화는 프론트 휴리스틱으로 `실패(0점)` 처리하도록 보정
- 앱 설정에 TTS 목소리 후보 조회/선택 UI를 추가하고, 선택 시 샘플 음성을 자동 재생하도록 연결
- 기존 `sign-to-speech` 응답의 `audioBase64`를 재사용해 미리듣기 재생 처리
- 스코프 아웃: 백엔드 채점 정책 및 TTS API 스펙은 변경하지 않음

## 주요 변경 사항
- MyPage에 로그아웃 메뉴와 확인 다이얼로그 추가
- `AuthSessionManager.clearSession()`으로 토큰과 활성 아이 정보를 함께 초기화
- 로그아웃 성공 후 Login으로 이동하며 보호 화면 백스택 제거
- 미구현 MyPage 메뉴는 비활성 상태로 표시
- 백엔드 logout API 연동은 모바일 refreshToken 정책 확정 전까지 스코프 아웃

## 스크린샷/로그

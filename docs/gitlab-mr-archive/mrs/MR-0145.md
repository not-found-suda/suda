## 주요 변경 사항
- **오탐지 최적화**: 잦은 오탐지(False Positive) 블랙홀 현상을 일으키던 '친구' 단어 제외 후 23클래스 축소 재학습
- **연속 추론 안정화**: Confidence Threshold 0.8 세팅 및 손 유실 시 15프레임(0.5초) 유예(Grace Period) 로직 도입
- **비수지기호 및 문장 누적**: 실시간 단어 버퍼링과 눈썹 델타값 기반 `(의문문)` 판별 파이프라인 통합
- **모바일 이식 세팅**: Android용 Float16 TFLite 모델 변환 (Softmax 레이어 삽입), Python 환경 TFLite 로컬 사전 검증 완료
- **데이터 파이프라인 통합**: S3 및 로컬 ZIP 기반 332차원 랜드마크 데이터 추출 파이프라인 코드(`sign2text_tcn`) 통합 및 레거시 제거
- 변경하지 않은 것 / 스코프 아웃: 파이썬 모듈 의존성(안드로이드 개발 시에는 MediaPipe Tasks API로 별도 포팅 예정)

## Description

## 요약
- 온라인 STT 오디오 업로드 포맷을 `m4a/audio/mp4`에서 `WAV/audio/wav`로 전환


## 작업 배경
- 관련 이슈: #S14P31A404-284
- 참고 문서/기획:
  - Clova CSR STT REST API 지원 포맷 기준에 맞춰 모바일 업로드 음원을 WAV로 정규화
  - 백엔드에서 `audio/mp4`를 차단하고 WAV 계열 MIME만 허용하도록 변경됨

## 주요 변경 사항
- 온라인 STT 녹음 방식을 `MediaRecorder` 기반 m4a에서 `AudioRecord` 기반 PCM WAV 생성으로 변경
- WAV 헤더 생성 유틸을 추가하고 16kHz/16-bit/mono PCM WAV 파일을 생성하도록 구현
- STT 업로드 MIME을 `audio/wav`로 변경
- WAV 헤더 생성 단위 테스트 추가
- 스코프 아웃: Gemini 아이 발화 보정 실패 대응은 백엔드 후속 작업으로 분리

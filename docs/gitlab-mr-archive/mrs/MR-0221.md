## 작업 배경

  - 자막 가독성 개선 및 검은 배경 제거

## 주요 변경 사항
- 소통 화면 상단 헤더/디버그 노출을 제거하고, 메인 카메라 영역 비중을 확대
- 카메라 오버레이에 상체 기준선(얼굴/어깨/중심선) 가이드를 추가하고 실촬영 기준으로 위치/크기 조정
- 자막 영역의 검은 배경 오버레이를 제거하고, 채팅 풍선에만 옅은 배경색을 적용
- 채팅 풍선 텍스트는 흰색으로 유지하고 발화 주체별 말풍선 스타일을 분리
- 스코프 아웃: 수어 인식 로직 및 백엔드 API 스펙은 변경하지 않음

## 스크린샷/로그
- UI 변경 스크린샷 첨부
  - 상체 가이드 오버레이 적용 화면
  - 자막 검은 배경 제거 및 말풍선 색상 반영 화면

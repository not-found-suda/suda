## Description

## 요약

- REST STT와 CLOVA Speech Streaming STT 결과를 같은 음성 파일 세트로 비교할 수 있는 로컬 runner를 추가했습니다.

## 작업 배경

관련 이슈: S14P31A404-423\
참고 문서/기획: STT streaming 전환 후 REST/Streaming 성능 비교 필요

## 주요 변경 사항

- 지정한 wav 폴더를 읽어 REST STT와 Streaming STT를 순차 호출하는 비교 runner 추가
- 각 방식의 인식 결과와 처리 시간을 CSV로 저장
- `STT_COMPARISON_ENABLED=false` 기본값으로 평소 서버 실행에는 영향 없도록 설정
- 비교 입력/출력 경로를 env로 지정 가능하게 구성

## 테스트 결과

- `.\gradlew.bat spotlessApply compileJava` 통과
- 로컬 wav 변환 파일로 비교 실행 예정
- 결과 CSV는 git workspace 밖 경로에 저장 가능

- 팀원/배포 환경에서 env를 설정하지 않으면 기본값 `false`로 동작하지 않습니다.

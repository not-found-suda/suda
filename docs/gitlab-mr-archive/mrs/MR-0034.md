## 주요 변경 사항
- **`FLAG_KEEP_SCREEN_ON` 도입**: 수어 인식 세션 중 기기가 잠자기 모드로 진입하는 것을 방지하여 사용자 흐름 끊김 제거
- **세션 상태 연동**: `ConversationViewModel`의 `sessionState`가 `Active`일 때만 화면 유지가 활성화되도록 동적 제어 구현
- **리소스 최적화**: `DisposableEffect`의 `onDispose`를 통해 세션 종료 또는 화면 이탈 시 즉시 플래그를 해제하여 불필요한 배터리 소모 방지
- **안정성 강화**: `Activity` 윈도우 플래그 방식 채택으로 별도 권한 요청 없이 시스템 레벨에서 안전하게 화면 유지 관리

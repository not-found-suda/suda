## Description

## 요약
- TensorFlow Lite 런타임을 LiteRT로 교체해 Android 16KB page size 경고를 해결했습니다.


## 작업 배경
- 관련 이슈: #S14P31A404-292
- Android Studio에서 `lib/x86_64/libtensorflowlite_jni.so`가 16KB page size를 지원하지 않는다는 경고가 발생했습니다.
- Android 15+ 타깃 앱은 Google Play 제출 시 16KB page size 지원이 필요합니다.

## 주요 변경 사항
- `org.tensorflow:tensorflow-lite` 의존성 제거
- `com.google.ai.edge.litert:litert` 의존성 추가
- 기존 `.tflite` 모델 파일과 추론 코드는 유지
- 스코프 아웃: Python 모델 변환 파이프라인 및 모델 구조 변경

## Description

## 요약
- 보호자가 이메일/비밀번호로 자체 회원가입을 할 수 있도록 화면과 API 연동을 구현

## 작업 배경
- 관련 이슈: #S14P31A404-231
- 참고 문서/기획:
  - 백엔드 회원가입 API: POST /api/v1/auth/signup
  - 현재 백엔드 SignupRequestDto는 email/password만 받으므로 이름 입력은 제외

## 주요 변경 사항
- 회원가입 화면 UI, 입력 검증, 로딩/오류 상태를 구현
- NoAuth Retrofit 기반 회원가입 API 계층과 DI를 추가
- 백엔드 정책과 동일한 비밀번호 검증 규칙을 클라이언트에 적용
- Login → Signup → Login 네비게이션을 연결하고 Login 중복 백스택을 방지
- 스코프 아웃: 이름 입력, 자동 로그인, 토큰 저장, 네이버 SDK 연동, 백엔드 수정

## 스크린샷/로그
- UI 변경: 보호자 회원가입 화면 추가

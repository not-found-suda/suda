```text
\[P3\] 카테고리 진행도 화면에서 아이 선택 후 홈으로 빠질 수 있습니다

카테고리 진행도 화면의 아이 선택 버튼은 `ChildSelect`로 이동하지만, 아이 선택 완료 후 복귀 로직은 현재 `previousRoute`가 `ReportHome`일 때만 리포트 화면으로 돌아가도록 되어 있습니다.

그래서 카테고리 진행도 화면에서 `Missing`/`NotFound` 상태로 `아이 선택하기`를 누른 뒤 프로필을 선택하면, `previousRoute`가 `ReportCategoryProgress`라서 현재 화면으로 돌아오지 않고 `Home`으로 fallback될 수 있을 것 같습니다.

카테고리 진행도 화면에서도 아이 선택 후 원래 리포트 화면으로 돌아오도록 `ReportCategoryProgress` route도 함께 처리하면 더 자연스러울 것 같습니다!!
```

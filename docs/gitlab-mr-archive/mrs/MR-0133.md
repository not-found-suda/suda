## Description

## 요약

아이별 퀴즈 기록을 리포트 화면에서 조회할 수 있도록 목록/상세 조회 API를 구현했습니다.

## 작업 배경

관련 이슈: S14P31A404-333\
참고 문서/기획: 

## 주요 변경 사항

- `GET /api/v1/children/{childId}/reports/sessions` 퀴즈 기록 목록 조회 API 추가
- `GET /api/v1/children/{childId}/reports/sessions/{sessionId}` 퀴즈 기록 상세 조회 API 추가
- 아이 프로필 소유권 검증 및 타 보호자/비활성 아이 접근 차단
- 정답률, 평균 별점, 문제별 답변 결과를 리포트 응답 DTO로 반환
- Path/Query 파라미터 타입 오류를 `400 VALIDATION_INVALID_INPUT`으로 처리

## 변경하지 않은 것 / 스코프 아웃

- 리포트 summary API는 이번 이슈 범위에서 제외했습니다.
- 카테고리별 리포트/취약 단어 API는 별도 이슈에서 구현 예정입니다.
- 별도 집계 테이블이나 집계 컬럼은 추가하지 않았습니다.

## 메모

- 목록 조회의 `page`는 API 명세 기준 `1`부터 시작하며, 내부 조회 시 Spring `PageRequest` 기준에 맞게 `page - 1`로 변환합니다.
- 퀴즈 기록 상세는 요청한 `childId`에 속한 `sessionId`만 조회되며, 다른 아이의 세션이면 `REPORT_NOT_FOUND`로 응답합니다.

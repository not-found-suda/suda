## Description

## 요약

- 마이페이지에서 사용할 소셜 계정 연동 상태 조회/네이버 연동/네이버 해제 API를 추가했습니다.

## 작업 배경

관련 이슈: S14P31A404-359\
참고 문서/기획: 모바일 유저 도메인 2순위 요청, 소셜 계정 연동 관리 화면

## 주요 변경 사항

- `GET /api/v1/users/me/social-accounts` 소셜 계정 연동 상태 조회 API 추가
- `POST /api/v1/users/me/social-accounts/naver` 네이버 계정 연동 API 추가
- `DELETE /api/v1/users/me/social-accounts/naver` 네이버 계정 연동 해제 API 추가
- 이미 다른 사용자에게 연동된 소셜 계정 충돌 처리 추가
- 마지막 로그인 수단인 소셜 계정 해제 방지 정책 추가

## 스크린샷/로그

- UI 변경 없음
- API 변경으로 Swagger 또는 응답 캡처 첨부 예정

## 메모

- 현재 실제 연동/해제 API는 네이버만 구현했습니다.
- 조회 응답은 `NAVER`, `KAKAO` provider 상태를 모두 내려주도록 구성해 추후 카카오 확장을 고려했습니다.
- 소셜 전용 계정에서 마지막 로그인 수단인 네이버 연동은 해제할 수 없도록 `409 SOCIAL_ACCOUNT_LAST_LOGIN_METHOD`로 처리했습니다.

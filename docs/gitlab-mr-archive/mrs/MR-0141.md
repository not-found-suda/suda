## 요약

* 카테고리별 리포트 요약 API를 추가하고, 리포트 요약의 최신 카테고리 조회 오류를 수정했습니다.

## 작업 배경

## 주요 변경 사항

- `GET /api/v1/children/{childId}/reports/categories` API 추가
- 완료된 퀴즈 세션 기준으로 카테고리별 정답률/평균 별점/최근 퀴즈일 집계
- 카테고리별 퀴즈 출제 단어 비율과 정답 경험 단어 비율 계산
- 리포트 요약 최신 카테고리 조회 시 누락된 `completed` 파라미터 바인딩 수정
- 변경하지 않은 것 / 스코프 아웃
  - 퀴즈 기록이 없는 카테고리는 초기 구현에서 응답에 포함하지 않습니다.
  - 단어 카드/따라 말하기 학습 기록은 현재 별도 저장 데이터가 없어 리포트 집계에 포함하지 않습니다.

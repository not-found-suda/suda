# 상세 기능 명세서 (Functional Specifications)

본 문서는 SUDA 프로젝트의 온디바이스(On-Device) 아키텍처를 기반으로, 실제 구현해야 할 상세 기능과 동작 로직을 정의합니다.

## 1. 공통 및 디바이스 제어 (COMM)

| ID | 기능명 | 기능 상세 설명 (On-Device 최적화) | 우선순위 |
| :--- | :--- | :--- | :--- |
| **COMM_001** | 실시간 소통 세션 제어 | 사용자가 '소통 시작/종료' 버튼을 누르면 단일 기기 내에서 로컬 카메라/마이크 리소스를 할당 및 해제한다. | P1 |
| **COMM_002** | 디바이스 권한 요청 | 앱 최초 실행 시 카메라, 마이크 권한을 요청하며 거부 시 스낵바를 통해 기기 자체 설정창으로 이동을 유도한다. | P1 |
| **COMM_003** | 화면 꺼짐 방지 (Wakelock) | 소통 세션이 유지되는 동안, 안드로이드 `FLAG_KEEP_SCREEN_ON` 플래그를 적용하여 화면이 절전 모드로 진입하지 않도록 처리한다. | P1 |
| **COMM_004** | 상태 및 지연 안내 | 기기 내부 1차 추론은 완료되었으나 Spring 서버(LLM/TTS) 통신 지연 시 로딩 인디케이터를 표시하며, 오프라인 상태 시 "네트워크 연결 불안정" 메시지를 토스트로 띄운다. | P2 |

## 2. 부모 (농인) -> 자녀 (청인) 발화 (PATOCH)

| ID | 기능명             | 기능 상세 설명 (On-Device 최적화) | 우선순위 |
| :--- |:----------------| :--- | :--- |
| **PATOCH_001** | 수어 영상 로컬 수집     | 전면 카메라 프레임을 외부 서버로 전송하지 않고, `CameraX`를 통해 기기 내에서 15~30fps로 자체 캡처한다. | P1 |
| **PATOCH_002** | 랜드마크 추출 및 로컬 추론 | 캡처된 프레임을 MediaPipe로 분석하여 좌표를 추출하고, 이를 **기기 내장 TFLite (LSTM) 모델**에 즉시 주입하여 원시 단어를 도출한다. | P1 |
| **PATOCH_003** | 인식 범위 이탈 안내     | 부모의 수어 동작이 전면 카메라 범위를 벗어날 경우, MediaPipe 좌표 유실을 감지하여 UI 테두리 색상 변경 등의 시각적 가이드를 노출한다. | P2 |
| **PATOCH_004** | 자연스러운 문장 변환     | 도출된 원시 단어 배열(텍스트)만 Spring Boot API Gateway로 POST 전송하여, Gemini API를 통해 자연어 문맥 교정을 수행한다. | P1 |
| **PATOCH_005** | 텍스트 음성 변환 (TTS) | 교정된 문장을 서버 측 CLOVA TTS로 변환한 뒤, 기기로 반환된 음성 데이터를 `MediaPlayer` 등을 활용해 지연 없이 재생한다. | P1 |
| **PATOCH_006** | 에코 캔슬링 (마이크 제어) | TTS 음성이 기기 스피커로 재생되는 0.5~1초 동안, 해당 소리가 마이크로 재입력되지 않도록 로컬 STT 대기 상태를 잠시 뮤트(Mute) 제어한다. | P1 |

## 3. 자녀 (청인) -> 부모 (농인) 발화 (CHTOPA)

| ID | 기능명 | 기능 상세 설명 (On-Device 최적화) | 우선순위 |
| :--- | :--- | :--- | :--- |
| **CHTOPA_001** | 로컬 STT 음성 인식 | 음성 스트리밍 서버 구축 없이, 안드로이드 내장 `SpeechRecognizer`를 활용하여 실시간으로 아이의 음성을 텍스트로 1차 추출한다. | P1 |
| **CHTOPA_002** | 영유아 발음 보정 | 1차 추출된 텍스트("옴마")를 Spring Boot 서버로 전송하여, 미리 정의된 LLM 프롬프트를 통해 영유아 특성을 고려한 2차 발음 보정("엄마")을 수행한다. | P1 |
| **CHTOPA_003** | 자막 통합 렌더링 | 보정되어 돌아온 아이의 텍스트와 부모의 텍스트를 한 화면에 말풍선 형태로 직관적으로 구분하여 렌더링한다. | P1 |
| **CHTOPA_004** | 주변 소음 안내 | 마이크 소음 레벨(dB)이 너무 높아 로컬 STT 인식이 실패하거나 지연될 경우, 부모가 상황을 인지할 수 있도록 화면 중앙에 아이콘 형태의 시각적 알림을 제공한다. | P2 |
